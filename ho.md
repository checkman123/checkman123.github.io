## Ho Web Application (React)

**Project description:** This is a presentation of HO application. HO application is a commercial application that host several stores within it. Figure 1 is the main page of HO application and some of its menu. Figure 2 is the store page in HO application. Each store has different color theme depending on the backend data selected by store admin and uses jQuery to the color theme. Figure 3 is a card inventory where user can scroll to each card to access the associated store with that card.

**Skills**
- UI/UX Design  
- HTML, CSS, JavaScript, JQuery, and Bootstrap framework
- Interaction between front-end and back-end
